Project Overview

CourierBot is a robotic courier system developed using a TurtleBot 4. It receives spoken delivery commands, navigates a mapped environment using Nav2, and confirms task completion through sound feedback. Our project integrates speech recognition using ROS 2 Whisper and includes a modular launch setup to coordinate speech, navigation, and task execution.
